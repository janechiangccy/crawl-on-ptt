# Random Password Generator

透過 string & random 進行高強度密碼的產出。

結合英文字母大小寫、數字、標點符號之字符，隨機生成一組自定義長度之密碼，

過程中需了解 string & random 之用法，如：

* string 

    - string.ascii_letters
    - string.ascii_lowercase
    - string.ascii_uppercase
    - string.digits
    - string.punctuation

* random

    - random.shuffle
    - random.sample
    - random.randint
